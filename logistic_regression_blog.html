Why Logistic Regression is Still Called Regression, Not Classification

When you first look at logistic regression, it feels confusing. The word regression sounds like predicting numbers, but logistic regression predicts categories like yes or no. So how does it manage that switch? The answer lies in one tiny curve called the sigmoid function.

In simple words, logistic regression starts exactly like linear regression. It builds a straight-line equation: z = b0 + b1x1 + b2x2 + … This z is just a number, it could be -10, 2.5, 8, or anything. If we stopped here, the model would behave just like linear regression and give continuous values. But classification needs something different, a probability between 0 and 1. That’s where the sigmoid function steps in.

The sigmoid function takes that z and converts it into a number between 0 and 1 using this formula: σ(z) = 1 / (1 + e^(-z)) You don’t need to memorize it, just know what it does. When z is a big positive number, sigmoid gives a value close to 1. When z is a big negative number, it gives a value close to 0. So basically, it squeezes every possible value into a range that we can treat as probability.

Let’s take a small example. Suppose we’re trying to predict whether a student will pass an exam. The model calculates z = 4 for one student and z = -3 for another. When we pass z = 4 into sigmoid, we get about 0.98, which means a 98% chance of passing. For z = -3, sigmoid gives about 0.05, or a 5% chance of passing. So the first student gets a “Yes” and the second a “No”. This simple curve just transformed a plain regression output into something we can read as probability.

That’s the real trick behind logistic regression. It’s still using the old regression base, but the sigmoid curve on top turns its output into probabilities. Then we just set a rule, usually if the probability is above 0.5, we call it “Yes,” otherwise “No.” That’s how a linear equation suddenly learns to make categorical predictions.

Once you see it like this, the name starts to make sense again. Logistic regression isn’t wrongly named, it’s simply a regression model that learned to think in probabilities.
