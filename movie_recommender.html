Building a Smart Movie Recommender with Content Similarity

This project focuses on creating a content-based movie recommendation system using the TMDB 5000 Movies dataset. The main goal was to design a system that suggests movies similar to the one a user selects, based purely on content information such as storyline, genres, cast, crew, and keywords.

The project began with merging and cleaning two datasets containing movie details and credits. Unnecessary columns were dropped, keeping only the ones that define the movie's content and theme. Text-based columns like genres, keywords, cast, and crew were originally stored as strings, so they were converted into lists using Python’s ast.literal_eval() function. For every movie, only the most relevant details were extracted — the top 3 actors, the director’s name, and the key genre and keyword tags.

To generate meaningful recommendations, all text information was merged into a single column called tags. These tags were converted to lowercase and passed through stemming using the PorterStemmer to ensure similar words (like “loved” and “love”) were treated the same. Using CountVectorizer from scikit-learn, each movie’s text data was transformed into a vector of 5000 key words, representing its unique fingerprint.

The next step was calculating cosine similarity between every pair of movies. This mathematical score measures how close two movie vectors are in meaning. When a user picks a movie, the system identifies its index and returns the top five most similar movies based on the highest similarity values.

Finally, a Streamlit web app was built to make the system interactive. The user can select any movie, and the app instantly displays five similar titles along with their posters fetched through the TMDB API. This gives an intuitive, real-world experience of how recommendation engines like Netflix or IMDb work.

The project demonstrates how NLP, vectorization, and machine learning can work together to deliver personalized user experiences. It’s a perfect example of how content understanding can drive smarter entertainment recommendations.
