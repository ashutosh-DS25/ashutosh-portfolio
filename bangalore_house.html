Building a Machine Learning Model to Predict Bangalore House Prices

This project focuses on predicting real estate prices in Bangalore using data-driven modeling. The dataset contains 13,320 property records with details such as location, size, total square feet, number of bathrooms, and price. The objective was to understand how various features influence housing prices and to create a model capable of giving quick, accurate predictions for users.

The first phase involved data cleaning and preprocessing. Columns like area_type, availability, society, and balcony were dropped due to redundancy or high missing values. Null values in location, size, and bath were filled using the mode or median. The size column was processed to extract the numeric number of bedrooms (BHK), and textual anomalies in total_sqft were handled by splitting ranges and converting them into integers.

To handle categorical imbalance, rare location values (less than 10 occurrences) were grouped into a single “other” category. Outliers were removed using logical rules such as eliminating records where price per square foot or area per BHK deviated excessively from the location mean. Additional features were engineered like sqft_per_bedroom and price_per_sqft for detecting inconsistent listings.

After outlier removal, the dataset reduced to approximately 10,000 records. The price column was scaled up to rupee values (multiplied by 1,00,000). Train-test split (80-20) was performed, and categorical encoding was handled using OneHotEncoder with a StandardScaler inside a pipeline.

Multiple regression models like Linear Regression, Ridge, and Lasso — were trained and compared. The Linear Regression model achieved an R² score of 0.887, which remained consistent across Ridge and Lasso, indicating strong and stable performance. The model was serialized using Pickle for deployment.

A Flask web application was developed where users can input property details like location, BHK, bathrooms, and total square footage. The app predicts house prices instantly based on trained model insights. The deployed model provides practical value for buyers and sellers to estimate property costs in different Bangalore locations.

This project demonstrates the full pipeline from raw CSV data to a functioning web app covering data cleaning, feature engineering, model building, and deployment. It’s a great example of how regression techniques can be applied effectively to real estate price prediction.
