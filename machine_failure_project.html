Tata Steel Machine Failure Prediction using Machine Learning

This project applies machine learning to predict potential machine failures in a steel manufacturing environment. The goal is to identify early warning signs before an actual breakdown occurs, helping to save maintenance costs and improve overall production efficiency.

The dataset contains information such as air temperature, process temperature, rotational speed, torque, and tool wear. Each record shows whether a failure happened or not. Using Python, exploratory data analysis was done to understand relationships between these parameters. Visualizations helped reveal patterns like how extreme torque or high temperature often lead to machine stress or breakdown.

After data cleaning and feature scaling, multiple models were tested including Logistic Regression, Random Forest, and XGBoost. Their performance was compared using metrics like accuracy, precision, and recall to balance detection and false alarms. Among them, the Random Forest model performed best with stable and interpretable results.

The project also highlights which factors most strongly influence failure probability through feature importance analysis. These insights can guide engineers to monitor specific conditions more closely in real operations.

Overall, the model acts as a preventive maintenance tool, allowing Tata Steel to predict and avoid unexpected machine failures. It shows how AI can turn raw sensor data into meaningful predictions and help industries move from reactive repairs to proactive decision-making.
